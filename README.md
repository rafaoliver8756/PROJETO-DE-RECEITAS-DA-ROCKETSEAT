Projeto Receita

Este foi meu segundo projeto utilizando apenas **HTML5 e CSS3**.
O projeto faz parte de um exercício prático da formação **Fullstack da Rocketseat**, onde desenvolvi uma página de receita aplicando conceitos fundamentais de estruturação e estilização.
Apesar de ser um projeto simples, foi muito importante para meu aprendizado e consolidação da base em desenvolvimento web.

 O que aprendi neste projeto

- Estruturação semântica com HTML
- Organização de conteúdo com listas
- Uso de `:root` para variáveis CSS
- Aplicação de `box-sizing`
- Espaçamentos e organização com margin e padding
- Estilização de títulos e hierarquia visual
- Inserção de ícones (coração no rodapé)
- Importação de fontes do Google Fonts
- Criação e estilização de footer

Tecnologias utilizadas:

- HTML5
- CSS3
